???+ question

    Internet 最早起源于（ ）。

    A、ARPAnet     B、以太网      C、NSFnet      D、环状网

??? success

    A、ARPAnet

    ARPANET（Advanced Research Projects Agency Network，高级研究计划署网络）是美国国防部在 1969 年 建立的计算机网络，是 Internet 的前身。

    它最初连接了 4 所大学的计算机，用于科研和通信。

    随后，ARPANET 不断扩展，逐渐演变成今天的互联网（Internet）。

    美国国家科学基金会（NSF）为了促进学术机构之间的合作和科学研究，于 1985 年 建立了 NSFNET。晚于 ARPANET



???+ question

    在请求分页系统中，页表中的辅存始地址是供（）参考的。

    A.页面置换        B.内存分配        C.页面换出        D.页面调入

??? success

    D.页面调入

    辅存始地址也叫：外存地址 或 磁盘块号。

    当访问某页时，页表项被查到该页 不在内存（标志位 = 0）

    触发 缺页中断

    系统要从 辅存（磁盘） 把这页读进内存

    这时候，就要根据 页表中记录的辅存始地址，去磁盘对应位置找到该页数据。

    页表中的辅存始地址是供 系统进行缺页调入时 参考的。

???+ question

    周期挪用方式常用于（ ）方式的输入/输出中。
    A、DMA       B、中断      C、程序传送     D、通道

??? success

    A、DMA 

    Direct Memory Access

    DMA 要想直接访问内存，就得使用 系统总线（包括地址线、数据线、控制线）。
    但这条总线 CPU 也在用
    
    CPU 和 DMA 轮流 使用总线
    每次 DMA 只挪用一个存储周期（cycle）
    CPU 暂停一个周期，然后继续执行
    整个系统交替进行
