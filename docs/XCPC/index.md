# 我不奢求自己永远都最幸运。

## 这个页面的起源

原本在半年前，我是想在自己的简历上留下一个 XCPC 的位置的。

所以在搭建博客的时候，也预留下了这么一个位置。最初甚至详实地记录了 vp，补题。我希望能在半年后，在这里写下我的奖项。

那段时间，我积极地写博客，做自己的模板，妄想着 XCPC 能成为我 OI 生涯有意义的延续。有时，我也能打出高光题，似乎一切在往好的方向进行。

## 不可控的意外

由于学校传统，组队主要是暑假前，教练根据 hdu 春季联赛排名来组。所以我没有找队友的这一个过程。

春季联赛是网络赛，所以公平性主要依赖自觉。当然这个我估计马上要改了，毕竟大家都知道现在科技在发展。

暑假集训，本来还要分名额，结果一个队友和恶魔做交易，开挂开出幻觉了，让自己原本就垃圾的基础越发垃圾。

不完全证据：

1. cf 赛时过题需要的东西（中国剩余定理），过了几个月一不小心说自己不会这个。
2. 赛时看到他用手机 gpt 生成东西，放在电脑旁边抄，还做贼心虚把屏幕瞥向（他自以为）别人看不到的地方。
3. 模拟赛时我和另一个队友讨论呢，他搁这发一个后缀为 utm_source=chatgpt.com 的东西（呵呵，上网查东西就是指用 gpt 查是吧）
4. 同一场 hdu 多校，“自己”过了一个线段树 + 树链剖分的题后，和队友说自己不会线段树，让队友写。
5. 同一场牛客多校，问我 ntt 是什么，结果半小时内把多项式题秒了，这个题要其它多项式运算，还需要推导。（呵呵，赛后还到大群里显摆，讨论什么 vector 模板好好用，真他妈三脚猫）

还有一些不能称之为证据，比如 cf 开挂开过头了，甚至力压我校一队选手，跑到橙名。

然而，每每没机会用挂的时候，performance 就拉完了，我感觉和我这紫名水平的都差的有点多吧。

我相信整个集训队，对他产生过质疑的人不会只有一两个，也不止一两次。更何况他的两个队友，还有前队友，高中有 OI 功底的选手，会感觉不到他有问题？

教练心软没把他开除，换队处理，他还厚颜无耻地，说自己没用 ai 作弊，说自己很讨厌这个决定，说觉得和原先的队友更契合，说换队会让伤害继续下去。

我请问，你的道歉除了嘴上说说，有什么分量？能不能让我们看出你付出些实质性的代价？我请问，你他妈直接和间接让多少人受损？你能不能自己滚回你的实力配的位置？

如果能预言到他失去开挂的机会后的场均 performance，事情发生时，我不会甚至没和另一位队友讨论。

只为了不要误伤，只为了不去扩大影响，只为了留一条退路。

打不出 performance，可以找个契合的借口换下队，重新做人。结果这都嫌不够好呢~

我似乎在前面的一段时间，说服自己，遗忘这个事情，直到前段时间，一个以前的同学问起，我的这个零基础队友和另一个同期零基础的水平，勾起了我们的讨论，和我痛苦的回忆。

这差距，现在只能说有点搞笑。我确实看到了一个努力的人，水平达到了一个很可观的高度。

另一个呢？投机取巧，偷奸耍滑，厚颜无耻，得寸进尺，葬送自己曾经也许有的那么一点的天赋，害了队友，害了那些原本值得更强队友的同学。

## 这件事情后的我

### 暂停追究

因为后面的 performance 无法在当时预言，同时教练很坚决地要给队友一次机会，以及我当时没有和太多人讨论这个事情，各方因素导致我的决策最终软弱了。但也难说这不是命运所指。

讨伐的事情我不再主动去干，我相信这种行为早晚会自己反噬，不需要我再操劳了。

另一方面，XCPC 无法方便区分同一场比赛各个队员的贡献，也导致了很多东西的模糊。其实一个队伍双核甚至单核的情况很多见。很难去奢求自己永远是最幸运的，抱上大腿的那个。

我不奢求自己永远都最幸运。

### 及时止损，退役

十天内我会结束我最后两场比赛，并退役。本来我的预期是改成不一定金，拿个银算了的；但是上周只打了个铜尾，现在也不敢再有什么预想了。

XCPC 的偶然性也导致，我不能保证再付出一年，就不会再碰到意外。今年平时看着没啥问题的二队，前三场也没打出金来，我想这是一个有意义的参考。
