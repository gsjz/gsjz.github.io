# sha256

省流：sha256 是通过神秘过程得到的哈希函数，共 256bit，通常用十六进制来表示，所以是 256/4 = 32 个字符。

## 简介

SHA-256（Secure Hash Algorithm 256-bit）是 SHA-2 哈希算法家族中最常用的一种，由美国国家安全局（NSA）设计，并由 NIST 发布为联邦信息处理标准（FIPS 180-4）。该算法将任意长度的输入映射为 **256 bit（32 字节）摘要**，通常以 **64 位十六进制字符串**的形式表示。

SHA-256 主要特点：

- 输出长度固定为 256 bit  
- 不可逆、抗碰撞能力强  
- 广泛用于密码学、安全协议、数字签名、区块链等场景  
- 性能适中，安全性远高于 MD5 与 SHA-1

SHA-256 是目前工业界主流的安全哈希算法之一。

=== "Linux CLI"

    ```bash
    echo -n "hello world" | sha256sum
    ```

=== "PowerShell"

    ```powershell
    Get-FileHash hello.txt -Algorithm SHA256
    ```
