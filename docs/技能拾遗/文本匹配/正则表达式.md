# 正则表达式

## 基本字符匹配

| 表达式  | 含义            | 示例     | 匹配结果      |
| ---- | ------------- | ------ | --------- |
| `a`  | 匹配字符 a        | `a`    | a         |
| `.`  | 匹配任意单个字符（除换行） | `a.c`  | abc / a1c |
| `\.` | 匹配字面量 `.`     | `a\.c` | a.c       |
| `\\` | 匹配反斜杠         | `\\`   | \         |


## 字符类

| 表达式      | 含义               | 示例             |
| -------- | ---------------- | -------------- |
| `[abc]`  | 匹配 a / b / c 中任一 | `[abc]`        |
| `[^abc]` | 匹配非 a / b / c    | `[^abc]`       |
| `[a-z]`  | 匹配小写字母           | `[a-z]`        |
| `[A-Z]`  | 匹配大写字母           | `[A-Z]`        |
| `[0-9]`  | 匹配数字             | `[0-9]`        |
| `\d`     | 数字               | 等价 `[0-9]`     |
| `\D`     | 非数字              |                |
| `\w`     | 单词字符             | `[A-Za-z0-9_]` |
| `\W`     | 非单词字符            |                |
| `\s`     | 空白字符             | 空格 / tab / 换行  |
| `\S`     | 非空白字符            |                |


## 数量限定符

| 表达式     | 含义      | 示例       | 匹配              |
| ------- | ------- | -------- | --------------- |
| `*`     | 0 次或多次  | `a*`     | "" / a / aaa    |
| `+`     | 1 次或多次  | `a+`     | a / aaa         |
| `?`     | 0 或 1 次 | `a?`     | "" / a          |
| `{n}`   | 恰好 n 次  | `a{3}`   | aaa             |
| `{n,}`  | 至少 n 次  | `a{2,}`  | aa / aaa        |
| `{n,m}` | n 到 m 次 | `a{2,4}` | aa / aaa / aaaa |


## 位置锚点

| 表达式  | 含义    | 示例         |
| ---- | ----- | ---------- |
| `^`  | 行首    | `^abc`     |
| `$`  | 行尾    | `abc$`     |
| `\b` | 单词边界  | `\bword\b` |
| `\B` | 非单词边界 |            |



## 分组与或

| 表达式       | 含义    | 示例        |      |
| --------- | ----- | --------- | ---- |
| `(abc)`   | 捕获分组  | `(ab)+`   |      |
| `(?:abc)` | 非捕获分组 | `(?:ab)+` |      |
| `a\|b`    | 或     | `cat      | dog` |



## 贪婪

| 表达式   | 含义    | 示例      |
| ----- | ----- | ------- |
| `.*`  | 贪婪匹配  | `<.*>`  |
| `.*?` | 非贪婪匹配 | `<.*?>` |

## 反向引用

| 表达式  | 含义        | 示例         |
| ---- | --------- | ---------- |
| `\1` | 引用第 1 个分组 | `(a)\1`    |
| `\2` | 引用第 2 个分组 | `(a)(b)\2` |


## 常见使用模式

| 目标          | 正则                        |
| ----------- | ------------------------- |
| 整数          | `^-?\d+$`                 |
| 浮点数         | `^-?\d+(\.\d+)?$`         |
| 邮箱（简单）      | `^[\w.-]+@[\w.-]+\.\w+$`  |
| IP v4       | `^(\d{1,3}\.){3}\d{1,3}$` |
| 16进制        | `^[0-9a-fA-F]+$`          |
| 中文（Unicode） | `[\u4e00-\u9fa5]`         |
| HTML 标签     | `<[^>]+>`                 |
