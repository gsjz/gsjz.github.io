# callout

| Type       | 常见语义 | 典型用途       |
| ---------- | ---- | ---------- |
| `note`     | 备注   | 普通说明       |
| `abstract` | 摘要   | TL;DR / 总结 |
| `info`     | 信息   | 背景说明       |
| `tip`      | 技巧   | 小技巧        |
| `success`  | 成功   | 正确做法       |
| `question` | 疑问   | 提问 / 思考    |
| `warning`  | 警告   | 注意事项       |
| `failure`  | 失败   | 错误示例       |
| `danger`   | 危险   | 严重后果       |
| `bug`      | Bug  | 缺陷说明       |
| `example`  | 示例   | 示例代码       |
| `quote`    | 引用   | 引用内容       |
