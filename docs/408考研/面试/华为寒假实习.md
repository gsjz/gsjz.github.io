# 某大厂面试

## 投递

其实有点神秘。当时 HR 在之前请客的群里说想明年上半年实习的联系他，我还有最后一场 CCPC 打完就退役，于是联系一下。

自己先是官网上把信息填了，以为简历只需要这些就行了。后面 HR 微信问我有没有自己的简历，需要了解一下我。

然而我还啥经历都没有，就很尴尬。HR 还说需要有 AI 方面的经历，那我更没有了。

接着又把我推到他同事那里，给我打电话。大致得知目前只有 AI 工程师这个岗位有 headcount。

我本来觉得算了，后面又想了一下，如果本科只留一年时间做科研，那只能做热门点的 AI 了。

所以这个岗位对我来说还算是 OK 的，只不过自己还不太符合要求。

因为银奖直接推进到面试了，只能硬着头皮试试看了。

## 技术面

面试官看上去比较年轻，说自己是做 AI 的。

最后总结说我 AI 还在入门阶段。但还是给通过了。听说实际上通过里也有细分，如果低的话主管面比较可能挂。

用的 wemeeting 双机位。开始前这个软件没有什么测试入口，面试时一开共享就掉线。

面试官有提说因为调试设备多花了 10min，也就是说原本计划是 30min。

### 自我介绍

绩点方面我说自己公共的数学课期末都是 95 左右，然后专业课是一些比较基础的，像信息论基础、计算机组成基础、面向对象之类的，目前接触过 Java, Python, C。

说了一下自己在竞赛中发挥比较多的的是凸优化、Lagrange 插值等，有一些 blog 的产出。目前找了导师，准备转向做异常检测方向的科研。

面试官问我为什么要转做这个方向，认为异常检测不是企业里比较热门的，是作为入门还是如何；以及为什么不准备继续弄竞赛，觉得这个竞赛的比较多做编译相关的。<del>其实还有一段话因为网络问题我没听到。</del>

我回答了一下说继续打 XCPC 的话感觉边际收益比较小，希望积累一些别的方向的能力，但是比较对口的 TCS 门槛过高，学校里也不太有这方面的导师。
因为后面还要留时间考研，能留给科研的时间只有一年，并且 AI 是现在的大趋势，想积累一些这方面的经验。

面试官随后拷打了一下最近有没有读论文。我答得比较烂，问我这个论文的指标是什么也回答不上来。

### ML 八股

接着考了 ML 八股。

supervised learning 和 unsupervised learning 的区别？

我回答说前者是预先知道答案的，后者不是，比如说一些聚类、相关内容推荐。顺便说了还有一个并列的强化学习，是从环境里接受反馈。
面试官还问我知不知道元学习，我说还不知道。

过拟合和欠拟合？

我挺记得之前在 kaggle 上分 train, valid, test 集的过程的，就说的是一些优化算法会依赖 valid 的结果做迭代优化，
过拟合就是在 valid 上表现良好，但是在 test 上表现不好。欠拟合说就是像一些复杂函数最后只拿线性函数来拟合。

机器学习常见架构了解哪些？我说自己听过 CNN, RNN, Transformer。

它们主要的层是什么？我糊了一下 CNN 的卷积是什么意思，和图像处理共通；然后 RNN, Transformer 只知道它们感性上的对比，Transformer 是 attention。面试官追问，提示我三个字母，我还没反应过来。

### 手撕

最后手撕，说有三个选项，前两个都是说调包跑通即可，我当时听着都要放弃了，幸好最后还有个 leetcode，稀疏矩阵乘法优化，秒了。

## 主管面

对我来说难度最大的是这个时间定在早上九点，然而我一个学期没早八，有点起不来。

面试官看上去像中年领导。

强调了一下不面技术，考察综合素质。

前一天背了“以用户为中心，以奋斗者为本，长期艰苦奋斗”，结果没问。基本只问了竞赛的准备过程和赛时情况。

虽然自己对队友并不是很满意，但是面试只能把队友也往好了说，体现自己作为队长对队伍的领导。

准时 30min 结束。

## 总结

09:53 收到感谢短信，按照网上经验，当天 23:45 会收到反馈邮件，次日 10:53 出面试结果。
